# Past presentations

Frozen, self-contained archives of each talk as delivered. Every folder has its own
`index.html` **and** its own `assets/` copy, so an archived deck renders and deploys
independently of the current root deck — future edits to the root can never break history.

To view one: open its `index.html` in a browser, or (once deployed) visit
`https://jolow99.github.io/ai-delegated-deliberation-slides/presentations/<folder>/`.
